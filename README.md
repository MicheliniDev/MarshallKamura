# Marshall Kamura
A simple modded version of the boss Kamura, of the game Inari, reworking him to be more aggressive, have custom combos
A rework for Skarrsinger Karmelita, one the major bosses in Hollow Knight Silksong, that makes her extraordinarily more difficulty and aggressive. Made for players who like the challenge that Karmelita brings. <br>

# Installation

The mod can be installed manually by:

1. Downloading the last stable version of BepInEx in https://github.com/BepInEx/BepInEx/releases
2. Extracting BepInEx to the game's root folder (the one with Inari.exe).
3. Starting the game to generate the necessary files and folder for BepInEx.
4. Getting the mod dll in the <a href="https://github.com/MicheliniDev/MarshallKamura/releases">Releases Page</a>
5. Placing the dll file in the BepInEx/plugins folder.

<b>Make sure that `HideManagerGameObject` is set to true on BepInEx/config/bepinex.cfg</b>
